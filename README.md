## 🕵️ Room Overview

During a crime-scene sweep, investigators discovered a mysterious pen drive in the victim's pocket.

Your task was to analyze the provided forensic disk image (E01), recover deleted artifacts, crack hidden secrets, and uncover the truth behind the handler's final message.
